# 📖 Guía de Pokédex del Entrenador

## Descripción
Este proyecto es una aplicación web temática inspirada en Pokémon.
Funciona como una **guía personal**, donde el entrenador puede consultar y registrar su progreso, explorar regiones y mapas, publicar retos y mantenerse informado con novedades sobre eventos.

---
## Contenido de la página
- **Pokédex personal**: listado de los Pokémon atrapados por el entrenador.
- **Regiones y mapas**: apartados visuales para ubicar las zonas a explorar.
- **Progreso del entrenador**: barras de avance que muestran el nivel de progreso en diferentes aspectos.
- **Formulario de retos**: permite publicar desafíos a otros entrenadores con datos como nombre, región, cantidad de Pokémon y fecha del combate.
- **Noticias**: sección para consultar novedades y ver contenido multimedia relacionado.

---
## Características
- Interfaz responsiva con Bootstrap 5.
- Diseño limpio y centrado para facilitar la consulta.
- Formularios simples y funcionales con confirmación visual.
- Integración de videos en la sección de noticias.
- Organización clara por secciones temáticas.
